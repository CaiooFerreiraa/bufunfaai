# Branches e Commits

## Branches

- `main`: producao
- `develop`: integracao
- `feature/*`: novas features
- `fix/*`: correcoes
- `hotfix/*`: correcoes urgentes
- `chore/*`: manutencao tecnica
- `docs/*`: documentacao

## Commits

Padrao `Conventional Commits`:

- `feat: adiciona base do modulo de autenticacao`
- `fix: corrige renovacao do refresh token`
- `chore: configura github actions`
- `docs: adiciona adr inicial`
